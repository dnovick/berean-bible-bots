# BBH — Additional Resources

Supplementary flashcard decks and reference material for Biblical Hebrew students
working through Pratico & Van Pelt's *Basics of Biblical Hebrew*.

These resources go beyond the chapter-by-chapter material and can be used alongside
or after completing the BBH course.

---

## Import Instructions

Each deck is provided in three formats:

- **`.md`** — human-readable reference with answer key
- **`.txt`** — Anki import file (File → Import, tab-separated)
- **`-fd.txt`** — Flashcards Deluxe import file

For Anki: all decks use tags so you can filter by chapter (`bbh-ch2` … `bbh-ch35`),
by stem (`stem-qal`, `stem-hiphil`, etc.), or by conjugation type.
