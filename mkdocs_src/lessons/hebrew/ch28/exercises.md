# Ch28 — Hophal Strong: Exercises

[← Back to lesson](index.md)

| Exercise | Description |
|---|---|
| [Function Sort](exercises/ch28-function-sort/index.md) | Classify 25 attested Hophal forms from Genesis, Exodus, Leviticus, and Numbers by semantic function (Physical Transfer / Legal Formula / Speech Reporting / State Change / Simple Action). Builds on BBH Chapter 28 (Hophal Strong Verbs). |
| [Hophal Hiphil Contrast](exercises/ch28-hophal-hiphil-contrast/index.md) | An 18-item drill contrasting Hiphil and Hophal forms drawn from the same roots, illustrating the causative-passive relationship that defines the Hophal. Builds on BBH Chapter 28 (Hophal Strong Verbs).  The exercise is organized in four parts: - **Part A** (items 1–7): Motion/transfer roots — all Hophal; students identify the passivized Hiphil action - **Part B** (items 8–11): Root <span class="heb">מוּת</span> — Hiphil/Hophal pairs with morphological contrast (vowel under ה distinguishes stems) - **Part C** (items 12–15): Root <span class="heb">נָגַד</span> — Hiphil (person as subject) vs. Hophal (information as subject) - **Part D** (items 16–18): Root <span class="heb">נָכָה</span> — Hiphil and Hophal with no common Qal counterpart |
| [Hophal Paradigm Drill](exercises/ch28-hophal-paradigm-drill/index.md) | Write all 26 inflected forms of מות (Hophal) from memory. No Imperative — passives cannot be commanded. |
| [Passage Exercise](exercises/ch28-passage-exercise/index.md) | Students identify and parse Hophal (passive causative) verbs across three passages: Genesis 22:20 (narrative report formula), Exodus 5:14–16 (oppression of Israel), and Numbers 35:16–18, 30–31 (death penalty statutes). The exercise includes 7 Hophal verbs and 4 distractors (2 Niphal, 2 Qal) that resemble Hophal forms. Students identify the stem, parse Conjugation/PGN/Root, and supply the corresponding Hiphil meaning (since Hophal = passive of Hiphil). |
| [Stem Id Drill](exercises/ch28-stem-id-drill/index.md) | *BBH Chapter 28 · Hophal Strong Verbs*  A 24-item identification drill mixing Qal, Niphal, Hiphil, and Hophal forms of strong roots (6 per stem), interleaved. Uses roots קטל (paradigm), שׁמר, כבד, and שׁמד. Students identify the stem, parse the conjugation and PGN, and translate each form. Items 16 and 20 share the identical spelling הַשְׁמֵר (Hiphil Imperative 2ms = Hiphil Inf. Absolute), reinforcing the ambiguity unique to strong roots. The Hophal has no Imperative; item 18 uses the Hophal Infinitive Construct in its place.  | File | Format | |---|---| | [ch28-stem-id-drill.md](ch28-stem-id-drill.md) | Static reference with answer key | | [ch28-stem-id-drill.html](ch28-stem-id-drill.html) | Interactive (fillable, ▶ Answer per row) | | [ch28-stem-id-drill.pdf](ch28-stem-id-drill.pdf) | Fillable PDF | |
