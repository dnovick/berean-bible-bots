# Pael Stem Drill

*Chapter 19 — The Pael Stem*

[Full Screen (Interactive)](ch19-pael-stem-drill.html){.md-button .md-button--primary}  [Print / PDF](ch19-pael-stem-drill.pdf){.md-button}  [Markdown](ch19-pael-stem-drill.md){.md-button}

A 20-item parsing drill covering all conjugations of the **Pael stem** (D stem) using verbs from Daniel and Ezra. Students identify conjugation, root, PGN, and translation for each form.

## Coverage

| Conjugation | Items | Verbs Used |
|---|---|---|
| Perfect | 1, 4, 5, 6, 9, 13, 16, 18 | שׁבח (3ms, 1cs, 3mp), ברך (3ms), חוה (3ms), שׁבח (1cp), קבל (3ms), שׁלח (3ms) |
| Imperfect | 2, 7, 10, 12, 15, 17, 20 | שׁבח (3ms, 3fs/2ms), ברך (3ms), חוה (3ms, 1cp), קבל (3mp), רבה (3ms) |
| Active Participle | 3, 8, 11 | שׁבח (ms), ברך (ms), חוה (ms) |
| Infinitive | 14 | שׁבח |
| Passive Participle | 19 | קטל (ms, model root) |

## Sample Questions

**Q1.** 1 שַׁבַּח &#9654; Answer
> **A:** Conj: Perfect &nbsp;|&nbsp; Root: שׁבח &nbsp;|&nbsp; PGN: 3ms &nbsp;|&nbsp; Trans: he praised (patach + dagesh-tsere in ב = Pael perfect 3ms)

**Q2.** 2 יְשַׁבַּח &#9654; Answer
> **A:** Conj: Imperfect &nbsp;|&nbsp; Root: שׁבח &nbsp;|&nbsp; PGN: 3ms &nbsp;|&nbsp; Trans: he will praise (יְ- prefix = Pael imperfect; contrast Peal יִ-)

**Q3.** 3 מְשַׁבַּח &#9654; Answer
> **A:** Conj: Participle (active) &nbsp;|&nbsp; Root: שׁבח &nbsp;|&nbsp; PGN: ms &nbsp;|&nbsp; Trans: praising (מְ- prefix + dagesh-tsere in ב = Pael active participle; Dan. 2:23)
