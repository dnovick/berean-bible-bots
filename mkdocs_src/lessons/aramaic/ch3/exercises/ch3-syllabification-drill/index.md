# Syllabification Drill

*Chapter 3 — Syllabification*

[Full Screen (Interactive)](ch3-syllabification-drill.html){.md-button .md-button--primary}  [Print / PDF](ch3-syllabification-drill.pdf){.md-button}  [Markdown](ch3-syllabification-drill.md){.md-button}

20-item drill covering syllable division, syllable type identification (open/closed), vowel class, and stress placement for common Aramaic words. Students divide each word into syllables, label each syllable type, identify the stressed syllable, and note any special features (quiescent letters, compensatory lengthening, Dagesh Forte).

## Coverage

| Category | Count |
|---|---|
| Basic division (open/closed) | 8 |
| Dagesh Forte doubling | 4 |
| Guttural / quiescence features | 4 |
| Accent and vowel reduction | 4 |
| **Total** | **20** |
