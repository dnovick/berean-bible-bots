# Syllable Division

*Chapter 3 — Syllabification and Pronunciation*

Divide 20 Hebrew words into syllables, identify syllable types, mark stress, and identify any Qamets Hatuf.

## Coverage

| Category | Description | Items |
|---|---|---|
| Open syllable (O-O) | Both syllables open; long vowels throughout | דָּבָר, בְּרֵאשִׁית, נָבִיא, תּוֹרָה, אֲדֹנָי |
| Open-Closed (O-C) | Mixed structure; stress on closed ultima | מֶלֶךְ, בְּרִית, שָׁבַת, שָׁלוֹם, חֶסֶד, קֹדֶשׁ, אֶרֶץ, כֹּהֵן |
| Closed-Closed (C-C) | Both syllables closed; Dagesh Forte context | מִשְׁפָּט |
| Three+ syllables | Multi-syllable words; stress on ultima | אֱלֹהִים, שָׁמַיִם, הָאָרֶץ |
| Five syllables | Place name; all open except ultima | יְרוּשָׁלַיִם |
| Single syllable | One closed syllable | עַם |
| Guttural composite shewa | Hateph vowel instead of simple vocal shewa | אֱלֹהִים (ֱ), אֲדֹנָי (ֲ) |
| Silent shewa | Closes syllable before a consonant cluster | יִשְׂרָאֵל, מִשְׁפָּט |
| Mater lectionis | Yod/waw/aleph/he as vowel letters | בְּרֵאשִׁית, תּוֹרָה, שָׁלוֹם, נָבִיא |

## Sample Questions

**Q1.** 1 אֱלֹהִים
> **A:** ✓ אֱלֹהִים אֱ-לֹ-הִים O-O-C הִים* — Hateph seghol (guttural); final mem closes syl. 3

**Q2.** 2 בְּרֵאשִׁית
> **A:** ✓ בְּרֵאשִׁית בְּ-רֵא-שִׁית O-O-C שִׁית* — Vocal shewa (word-initial); aleph mater in syl. 2; yod mater in syl. 3; taw closes

**Q3.** 3 הָאָרֶץ
> **A:** ✓ הָאָרֶץ הָ-אָ-רֶץ O-O-C רֶץ* — Article הָ open; aleph (guttural) opens syl. 2; both qamets = long ā (open syllables)
