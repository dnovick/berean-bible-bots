# Passage Exercise

*Chapter 14 — Qal Perfect Weak Verbs*

Parse Qal Perfect weak-root verbs from Genesis 1–4 and Exodus 1–2.
All target forms are weak-root Qal Perfects; a few strong forms are included as distractors.

## Coverage

| Weak Class | Count |
|---|---|
| III-ה | 4 |
| III-א | 3 |
| III-guttural | 3 |
| I-guttural | 3 |
| I-נ | 2 |
| I-י | 2 |
| Biconsonantal | 2 |
| Geminate | 1 |

## Sample Questions

**Q1.** 1 הָיָ֤ה
> **A:** ✓ הָיָ֤ה 3 s m היה III-ה Stative — "was" describing the serpent

**Q2.** 2 אָמַ֣ר
> **A:** ✓ אָמַ֣ר 3 s m אמר I-guttural Simple Past — perfect qamets unchanged; ר at R3 also guttural

**Q3.** 3 אָכַ֑לְנוּ
> **A:** ✓ אָכַ֑לְנוּ 1 p c אכל I-guttural Simple Past — 1cp suffix נוּ-
