# Alphabet Drill

*Chapter 3 — The Alphabet and Pronunciation*

[Full Screen (Interactive)](ch3-alphabet-drill.html){.md-button .md-button--primary}  [Print / PDF](ch3-alphabet-drill.pdf){.md-button}  [Markdown](ch3-alphabet-drill.md){.md-button}

A 24-item drill covering all letters of the Greek alphabet. Students identify the letter name and its pronunciation/sound from the written lowercase (and occasional uppercase) form. The drill reinforces both letter recognition and the phonetic value of each letter, including final sigma and the gamma nasal.

## Coverage

| Item | Form | Answer |
|---|---|---|
| 1–24 | Lowercase Greek letter | Letter name + sound description |

## Sample Questions

**Q1.** 1 α
> **A:** Alpha — "father" (long α) or "along" (short α); length varies

**Q2.** 2 β
> **A:** Beta — "b" as in "Bible"

**Q3.** 3 γ
> **A:** Gamma — "g" as in "gone" (hard g); but before γ, κ, χ, ξ it sounds like "ng" in "sing" — e.g., ἄγγελος = "ang-ge-los"
