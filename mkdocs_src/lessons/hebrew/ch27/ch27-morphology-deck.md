## How to Use

- **Front of card:** the pointed Hebrew form + reference
- **Back of card:** root + weak class + conjugation + person-gender-number + gloss
- **Tags** let you filter by weak class or conjugation in Anki

> **Note on III-aleph class (cards 11–15):** The original deck used מָצָא and קָרָא as exemplars, but neither root attests the required Hiphil conjugations in the Masoretic text (קָרָא has no Hiphil at all; מָצָא Hiphil is rare and lacks 3ms forms). The III-aleph class is now illustrated with **חָטָא** (H2398, "to sin"), which is the most robustly attested III-aleph Hiphil root in the OT and shows all five conjugation types clearly.
>
> **Note on יָסֵב (card 38):** A Hiphil imperfect 3ms of סָבַב is not attested in TAHOT as an unaffixed form; this card is labeled as a **paradigm form** to model the expected vowel pattern.

Study tip: drill by weak class first — master the I-guttural cards before mixing in Biconsonantal, etc.
