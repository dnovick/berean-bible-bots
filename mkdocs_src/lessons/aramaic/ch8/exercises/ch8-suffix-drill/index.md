# Suffix Drill

*Chapter 8 — Pronominal Suffixes*

[Full Screen (Interactive)](ch8-suffix-drill.html){.md-button .md-button--primary}  [Print / PDF](ch8-suffix-drill.pdf){.md-button}  [Markdown](ch8-suffix-drill.md){.md-button}

A 20-item drill covering pronominal suffixes from BBA Chapter 8 (*Basics of Biblical Aramaic*, Van Pelt). Each item presents an Aramaic noun or prepositional phrase with a pronominal suffix; the student identifies the base form, the suffix (person, gender, number), and translates.

## Coverage

| Suffix Types Covered |
|---|
| Suffixes on ms and fs singular nouns |
| Suffixes on mp plural nouns |
| Suffixes on independent prepositions (עַל, מִן, עִם, קֳדָם) |
| Suffixes on prefixed prepositions (לְ-) |
| All persons: 1cs, 1cp, 2ms, 2mp, 3ms, 3mp, 3fs |
