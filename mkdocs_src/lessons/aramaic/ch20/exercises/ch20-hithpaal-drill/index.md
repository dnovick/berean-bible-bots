# Hithpaal Drill

*Chapter 20 — The Hithpaal and Ithpaal Stems*

A 20-item parsing drill covering the **Hithpaal (Ithpaal)** stem (Dt stem — reflexive/passive of the Pael) using verbs from Daniel and Ezra. Students identify stem (noting metathesis where applicable), conjugation, root, PGN, and translation for each form.

## Coverage

| Conjugation | Items | Verbs Used |
|---|---|---|
| Perfect | 1, 2, 4, 6, 7, 9, 10, 13, 15, 17 | שׁכח (3ms, 3mp, 3fs), חשׁב (3ms), נדב (3ms, 1cp), מלא/מלי (3ms), שׁוה (3ms), קטל (3ms, 3mp — model) |
| Imperfect | 3, 8, 12, 16, 19 | חבל (3ms), רמם/רום (3ms), חשׁב (3fs/2ms, 1cs), קטל (3mp — model) |
| Participle | 5, 11, 14, 20 | שׁכח (ms, fp), נדב (mp), קטל (ms — model) |
| Infinitive | 18 | קטל (model) |

## Sample Questions

**Q1.** 1 הִשְׁתַּכַּח &#9654; Answer
> **A:** Stem: Hithpaal (metathesis) &nbsp;|&nbsp; Conj: Perfect &nbsp;|&nbsp; Root: שׁכח &nbsp;|&nbsp; PGN: 3ms &nbsp;|&nbsp; Trans: it/he was found (Dan. 5:11: "an extraordinary spirit was found in him"; metathesis: שׁ+תִ → שְׁתַּ; dagesh forte in כ = D-stem)

**Q2.** 2 אִתְחַשַּׁב &#9654; Answer
> **A:** Stem: Hithpaal (no metathesis — R1 = ח, not a sibilant) &nbsp;|&nbsp; Conj: Perfect &nbsp;|&nbsp; Root: חשׁב &nbsp;|&nbsp; PGN: 3ms &nbsp;|&nbsp; Trans: it was reckoned / considered (אִתְ- prefix + dagesh forte in שׁ (R2) = Hithpaal; no metathesis because R1 = ח)

**Q3.** 3 יִתְחַבַּל &#9654; Answer
> **A:** Stem: Hithpaal &nbsp;|&nbsp; Conj: Imperfect &nbsp;|&nbsp; Root: חבל &nbsp;|&nbsp; PGN: 3ms &nbsp;|&nbsp; Trans: it will be destroyed / harmed (Dan. 6:23: no wound was found on Daniel; יִתְ- = Hithpaal imperfect prefix; dagesh forte in ב (R2))
